___

## How to run project:

1. clone project
- `git clone `
2. create virtualenv
- `python3 -m venv venv`
3. activate virtualenv
- `source venv/bin/activate`
4. install requirements
 `pip install -r requirements.txt`
5. run project
- `python manage.py runserver`
